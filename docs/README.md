# thermounits Documentation

This directory contains detailed documentation for the `thermounits` package.

`thermounits` is a unit-safe thermodynamics and heat-transfer toolkit for Python. It provides:

- dimension-checked `Quantity` arithmetic,
- deterministic thermodynamic helper functions,
- fluid property models for water/steam and dry air,
- a complete CLI for quick engineering calculations.

## Documentation Map

- `installation.md` - installation, requirements, and environment setup
- `cli.md` - full command-line reference with examples
- `python-api.md` - Python API guide for units, thermo functions, and fluids
- `architecture.md` - package structure and design decisions
- `models-and-assumptions.md` - model validity, simplifications, and limitations
- `development.md` - local development, testing, linting, and release workflow

## Version

These docs target `thermounits` version `0.1.0`.
